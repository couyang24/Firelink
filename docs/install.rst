
Installation
============

.. note::

``Firelink`` is supported for 3.7+ and dependency is listed in ``setup/requirements.txt``


Developer Installation
-------------------------------------------

::

    git clone https://github.com/couyang24/Firelink.git
    cd Firelink
    pip install -e .

User Installation
-------------------------------------------

::

    git clone https://github.com/couyang24/Firelink.git
    cd Firelink
    pip install .
